# SnapMap
## A SYDE 322 Project

Using Google's Cloud Vision API, SnapMap allows you to upload a picture in the app to determine its location.

### Quick Start
```bash
git clone https://github.com/andrewjungg/snapmap.git
```

### Client Code
```bash
cd snapmap/client
npm i
```

### Server Code
```bash
cd snapmap/server
npm i
```
